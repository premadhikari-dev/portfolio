# Prem Adhikari – Developer Portfolio

Personal portfolio website built with Next.js and Tailwind CSS.

## 🚀 Live Website
https://premadhikari.com.np

## 🛠 Tech Stack
- Next.js
- React
- Tailwind CSS
- JavaScript

## ✨ Features
- Responsive design
- Project showcase
- Skills section
- Resume download
- Contact section

## 📂 Installation

```bash
git clone https://github.com/premadhikari-dev/portfolio.git
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
